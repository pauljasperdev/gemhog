# page.tsx

## Purpose

Login page that toggles between sign-in and sign-up forms. Manages authentication flow state to let users switch between creating a new account and logging into an existing one. Acts as the single entry point for all authentication-related user interactions.

## Exports

- `default LoginPage(): JSX.Element` - Login page with form toggling
