# card.tsx

## Purpose

Compound card component providing structured content containers. Includes Card (wrapper), CardHeader, CardTitle, CardDescription, CardAction, CardContent, and CardFooter sub-components. Supports default and small size variants with responsive styling and proper slot-based composition.

## Exports

- `Card({ className, size, ...props }): JSX.Element` - Main card container
- `CardHeader({ className, ...props }): JSX.Element` - Card header section
- `CardTitle({ className, ...props }): JSX.Element` - Card title element
- `CardDescription({ className, ...props }): JSX.Element` - Card description text
- `CardAction({ className, ...props }): JSX.Element` - Card action area
- `CardContent({ className, ...props }): JSX.Element` - Card body content
- `CardFooter({ className, ...props }): JSX.Element` - Card footer section
