## Decisions Made

- RESEND_API_KEY is required (not optional) because Resend is the production
  email provider; the optional status was a leftover from migration
- Created makeEmailLayers factory rather than just removing ternary, because the
  layer composition was duplicated in two files (email-layers.ts and
  subscriber.ts)
- Used re*local_dev_placeholder as local dev default -- passes startsWith("re*")
  validation but is obviously not a real key
- makeEmailLayers detects the dev placeholder key and uses EmailServiceConsole
  -- centralizes fallback logic in one place rather than ternaries at callsites
- Added plain text versions to email templates as part of the SendEmailParams
  refactor for better email client compatibility
